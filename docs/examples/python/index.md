## Overview

The following example uses the [pygnmi](https://pypi.org/project/pygnmi/) python module to talk to gNMI
devices.

```shell
python3 pygnmi.py
```

The following will simply run the pygnmi.py file using python3 to get the
openconfig interfaces.
